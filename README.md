🦊 FoxShop – React Online Store

FoxShop is an online store built with React, powered by Vite and styled with SCSS. This project showcases UI structure, state management, and shopping cart logic.

---------------------------------------------------------------------------------------------


Features


🛍 Product listing

🔍 Search functionality

🛒 Shopping cart (add/remove/change quantity)

🖱 Scroll lock when cart or menu is open


! Tech Stack: React | Vite | SCSS | ESLint + Prettier


---------------------------------------------------------------------------------------------

!!! Notes

Product links currently refresh the page instead of routing to separate product pages.
Product detail pages and full routing are not implemented yet.

---------------------------------------------------------------------------------------------
Getting Started

Clone the repository: git clone https://github.com/samobutniy-1/FoxShop-the-online-store

Install dependencies: npm install

Run development server: npm run dev

Open in browser: http://localhost:5173
